# 🏪 ТехноБазар - Магазин б/у бытовой техники

Полнофункциональный сайт для продажи б/у бытовой техники с корзиной, фильтрами, калькулятором доставки и админ-панелью.

## ✨ Функционал

### Для покупателей:
- 🏠 **Главная страница** - популярные товары, категории, преимущества
- 📦 **Каталог** - сетка товаров с фильтрами и поиском
  - Фильтры по категориям, цене, состоянию, бренду, году
  - Полнотекстовый поиск
  - Сортировка
- 🔍 **Карточка товара** - галерея фото, характеристики, добавление в корзину
- 🛒 **Корзина** - управление товарами, расчет доставки
- 🚚 **Калькулятор доставки** - расчет стоимости по расстоянию, этажу, сборке
- 📞 **Контакты** - телефоны, адрес, карта, форма обратной связи

### Для администратора:
- 🔐 **Админ-панель** (`/admin/`)
- ➕ Добавление/редактирование/удаление товаров
- 🏷️ Управление категориями
- ⚙️ Настройка контактной информации
- 🚚 Настройка цен на доставку и сборку
- 📥 Экспорт/импорт данных в JSON

## 🚀 Быстрый старт

### 1. Запуск локально

Просто откройте файл `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен npx)
npx serve

# PHP
php -S localhost:8000
```

### 2. Развертывание на хостинге

1. Загрузите все файлы на веб-сервер
2. Убедитесь, что доступен `index.html`
3. Админ-панель доступна по адресу `/admin/`

**Поддерживаемые хостинги:**
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
- Любой статический хостинг

## 📝 Управление данными

### Через админ-панель

1. Откройте `/admin/`
2. Используйте вкладки для управления:
   - **Товары** - добавление, редактирование, удаление
   - **Категории** - управление категориями
   - **Контакты** - телефоны, адрес, соцсети
   - **Доставка** - настройка цен
   - **Экспорт/Импорт** - резервное копирование

### Через JSON-файлы

Данные хранятся в `localStorage`, но можно подготовить начальные данные в файлах:

- `data/products.json` - товары и категории
- `data/config.json` - настройки магазина

## 🎨 Кастомизация

### Цвета
Измените CSS-переменные в `css/style.css`:

```css
:root {
    --primary: #2563eb;      /* Основной цвет */
    --secondary: #10b981;    /* Дополнительный */
    --accent: #f59e0b;       /* Акцентный */
    --dark: #1e293b;         /* Тёмный */
    /* ... */
}
```

### Логотип и название
Измените в `data/config.json` и на всех страницах HTML.

### Контактная информация
Редактируйте через админ-панель или в `data/config.json`.

- Данные хранятся в LocalStorage браузера
